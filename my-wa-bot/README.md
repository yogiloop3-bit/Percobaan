# My WA Bot

A simple WhatsApp bot using whatsapp-web.js.

## Installation

1. Clone the repository:
   ```bash
   git clone https://github.com/yogiloop3-bit/Bot-wa-triall.git
   cd Bot-wa-triall
   # cd my-wa-bot (if the bot code is in a subdirectory)
   ```

2. Install dependencies:
   ```bash
   npm install
   ```

2. Run the bot:
   ```bash
   npm start
   ```

3. Scan the QR code with your WhatsApp app.
